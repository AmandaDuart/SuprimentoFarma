# SuprimentoFarma

Projeto de gestão de estoque de medicamentos

# Como rodar a aplicação

### Clone o projeto
```
$ git clone https://github.com/AmandaDuart/SuprimentoFarma.git
```
### Instale as dependências
 ```
 npm install
 
```
### Execute a aplicação
 npm start
